# AeroPlus Deco

A technical dive decompression planner for **open-circuit** and **closed-circuit rebreather** diving, built on the **Bühlmann ZHL-16C** algorithm with gradient factors. Runs in any modern browser, installs as a home-screen app, works offline.

## About the app

AeroPlus Deco is developed by **Magnolia Blossom BV** (Doesburg, The Netherlands) for technical divers who want a clear, fast planner that runs anywhere without installation. The app calculates open-circuit and CCR profiles, tracks oxygen toxicity, supports gradient factor tuning (including the Andy Davis method for OC), and produces shareable plans you can send to a buddy via a single link.

The codebase is a single HTML file — no servers, no analytics, no telemetry. Your plans never leave your device unless you share them.

!!! warning "Important disclaimer"
    Always cross-check your decompression plans using a second tool or a qualified instructor. This app is for planning assistance only. Diving outside no-stop limits carries serious risks — incorrect decompression can result in injury or death. See the full [disclaimer](disclaimer.md).
