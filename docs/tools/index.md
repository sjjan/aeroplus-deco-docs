# Calculators

Three standalone tools that complement the main dive planner. Access via the **Tools** menu (☰ → Best mix / MOD / EAD / Gas blending).

## In this section

- **[Best mix](best-mix.md)** — derive the optimal O₂/He percentages for a target depth and PPO₂
- **[MOD / EAD / END](mod-end.md)** — analyze an existing mix: max operating depth, narcotic depth, density
- **[Gas blending](gas-blending.md)** — partial pressure blending with heat compensation

All three calculators open as full-screen overlays and operate independently of the active plan. Settings like salinity, altitude, and max PPO₂ are inherited from the current plan's configuration.

The **Best mix** and **MOD/END** calculators now pre-fill the depth field with the deepest segment in your active plan (if any), making it easy to check the gas you're using at your dive's max depth.
