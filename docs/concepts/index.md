# Concepts

These pages explain the decompression theory and terminology AeroPlus Deco uses, so you can read the plan output meaningfully and tune the settings to match your training and physiology.

You don't need to read them in order — each page stands alone. If you're new to gradient factors, start with [Gradient factors](gradient-factors.md). If you're trying to understand a specific number in the output, jump straight to the relevant page.

- **[Tissue saturation & GF99](tissue-saturation.md)** — Bühlmann compartments, M-values, and reading the GF99 chart
- **[TTS & deco stops](tts-and-deco-stops.md)** — what time-to-surface means and how the runtime schedule is built
